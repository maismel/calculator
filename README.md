## 2. How to run the app

Install dependencies:

npm install

Start the development server:

npm start

This will open the app in your browser automatically.

Build the production version:

npm run build

The optimized files will be generated in the dist folder.

## 3. Project Structure
src/ – source files: index.html, index.js, styles/ for CSS, and modules/ for JS logic.

dist/ – built files (generated after npm run build).

package.json – project info, dependencies, and scripts.

.gitignore – files/folders excluded from Git (e.g., node_modules/, dist/).

README.md – project documentation.
